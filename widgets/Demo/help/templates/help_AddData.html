Add Data Tool

Video Example [Exit EPA]

The Add Data tool allows online or local data to be added to the map. Tabs provide three different ways to add data to the EnviroAtlas map:

  • Search – provides the capability to search for data layers in EPA's GeoPlatform, the Federal GeoPlatform, or ArcGIS Online, Esriâ€™s collaborative cloud environment and add the layers to the EnviroAtlas interactive map.
  • URL – provides the capability to add dynamic map services, image services, feature services, or tiled services along with Open Source WMS and KML services to the EnviroAtlas interactive map from a known URL.
  • File – provides the capability to upload zipped shape files, CSV (comma-separated variables), GeoJSON, KML, or GPX files from a local drive to the EnviroAtlas interactive map.
