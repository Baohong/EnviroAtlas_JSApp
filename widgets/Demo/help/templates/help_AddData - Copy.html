Add Data Tool

The Add Data tool allows data from online sources to be added to the map. The two tabs in the Add Online Data pop-up provide three different ways to add online data to the EnviroAtlas map:

  • Search – provides the capability to search for data layers in ArcGIS Online, Esri’s collaborative cloud environment or the EPA's and add them to the EnviroAtlas interactive map.
  • Enter A URL – provides the capability to add dynamic map services, image services, feature services, tiled services along with Open Source WMS and KML services to the EnviroAtlas interactive map from a known URL.
  • Upload files – provides the capability to upload zipped shape files, CSV (comma-separated variables), GeoJSON, KML, or GPX files.
